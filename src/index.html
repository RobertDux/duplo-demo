Git

  • Start a new repository:
    • git init
    • git clone <repo_url>
  • Create a commit:
    • git commit -m "feat(page): add simple skeleton"
    • commit often
    • make small atomal commits
    • use sensible messages (NestJS example)
  • Push and pull:
    • git push to push your code to the online repo
    • git pull to pull code from the online repo
    • pull before starting new branches
  • Branching:
    • git switch -c branch_name to create and switch to a new branch
    • git merge branch_name to merge the branch_name into the current branch
    • Use feature branches to work on small features for easy collaboration
  • Other usefull commands and links:
    • git status to view status
    • git log to view commit history
    • GitHub Git Cheatsheet (PDF)
